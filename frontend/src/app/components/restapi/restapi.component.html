REST API

The ATAV data browser provides a REST API that users can programmatically query for variant information using variant, gene, or region search options. The API returns results in JSON format.

Note: the REST API has a daily rate limit to throttle the incoming requests in order to make sure they are running as efficiently as possible.

API Examples

Query by variant:

GET /api/variant/chr-pos-ref-alt

http://atavdb.org/api/variant/21-33040861-G-C


Query by gene:

GET /api/gene/hgnc_gene_name

http://atavdb.org/api/gene/TBK1


Query by region:

GET /api/region/chr:start-end

http://atavdb.org/api/region/2:166892071-166893071
